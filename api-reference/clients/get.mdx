# Get Client

Retrieve a specific client by its ID.

## Endpoint

```
GET /clients/:id
```

## Path parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the client
</ParamField>

## Request example

```bash
curl http://localhost:8000/clients/1
```

## Response

<ResponseField name="id" type="integer">
  Unique identifier for the client
</ResponseField>

<ResponseField name="name" type="string">
  Client's full name
</ResponseField>

<ResponseField name="email" type="string">
  Client's email address
</ResponseField>

### Success response (200)

```json
{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com",
  "phone": "+1-555-0123",
  "address": "123 Main St"
}
```

### Error response (404)

If the client doesn't exist:

```json
{
  "error": "not found"
}
```

## Use cases

- Retrieve client details for display
- Verify client existence
- Get client information before updating
- Fetch client data for reports

<Tip>
Check the HTTP status code. A 404 response means the client ID doesn't exist.
</Tip>
