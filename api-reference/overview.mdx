## Base URL

```
http://localhost:8000
```

## Authentication

The Simple API does not require authentication. All endpoints are publicly accessible.

## Response format

All responses use JSON format. Successful responses contain the requested data, while errors are formatted as:

```json
{
  "error": "not found"
}
```

## Status codes

| Code | Description |
| --- | --- |
| 200 | OK - Request succeeded |
| 201 | Created - Resource was created successfully |
| 404 | Not Found - Resource or endpoint doesn't exist |

## Rate limiting

No rate limiting is applied to the Simple API.

## Pagination

The Simple API does not support pagination. List endpoints return all available resources.

<Tip>
All endpoints follow the same pattern. Once you understand one resource type, you can apply the same operations to all others.
</Tip>
