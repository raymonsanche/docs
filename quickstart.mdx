Get the API running and make your first API calls in just a few minutes.

## Prerequisites

- Python 3.6 or higher
- A terminal or command prompt
- curl, Postman, or any HTTP client

## 1. Start the server

Navigate to your project directory and start the API server:

```bash
python3 main.py
```

You should see:

```
Server running on http://localhost:8000
Routes: /clients, /products, /categories, /services, /accounts
```

## 2. Create your first client

Send a POST request to create a new client:

```bash
curl -X POST http://localhost:8000/clients \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John Doe",
    "email": "john@example.com"
  }'
```

Expected response:

```json
{
  "id": 1,
  "name": "John Doe",
  "email": "john@example.com"
}
```

## 3. List all clients

Retrieve all clients:

```bash
curl http://localhost:8000/clients
```

Response:

```json
[
  {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
  }
]
```

## 4. Get a specific client

Retrieve a client by ID:

```bash
curl http://localhost:8000/clients/1
```

## 5. Update a client

Update an existing client:

```bash
curl -X PUT http://localhost:8000/clients/1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Jane Doe",
    "email": "jane@example.com"
  }'
```

## 6. Delete a client

Delete a client by ID:

```bash
curl -X DELETE http://localhost:8000/clients/1
```
