## Overview

Simple API is a lightweight RESTful API for managing clients, products, categories, services, and accounts. It provides standard CRUD operations over HTTP with JSON request and response bodies.

## Key features

- **Simple REST endpoints** - Standard HTTP methods for all operations
- **JSON-based** - All request and response bodies use JSON format
- **In-memory storage** - Fast, immediate operations with data that persists during server runtime
- **No dependencies** - Built with Python's standard library for lightweight deployment

## Resources

The API manages five resource types, each with full CRUD support:

- **Clients** - Manage customer information
- **Products** - Create and manage product inventory
- **Categories** - Organize products by category
- **Services** - Define and track services offered
- **Accounts** - Manage user and business accounts
